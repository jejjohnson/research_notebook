Each notebook is executed end-to-end with outputs embedded, so everything (figures, prints, tables) renders inline in the MyST docs site without re-execution.

Running locally¶

These notebooks depend on pyrox and its ML stack (JAX, NumPyro, equinox, matplotlib). The copies here carry their outputs, so MyST renders them without needing the kernel installed. To re-execute locally, install pyrox and the showcase deps separately:

pip install "pyrox @ git+https://github.com/jejjohnson/pyrox"
# or, in a fresh env:
pixi run -e jupyterlab lab
# then install pyrox in the kernel
